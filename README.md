# 🎮 Python Hangman Game

A fun console-based Hangman game built using Python.

## Features
- Multiple categories (Programming, Fruits, Animals)
- Random word selection
- ASCII Hangman graphics
- Input validation
- Score tracking (Wins & Losses)
- Play Again option

## Technologies Used
- Python

## How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Python-Hangman-Game.git
```

2. Open the project folder

```bash
cd Python-Hangman-Game
```

3. Run the game

```bash
python hangman.py
```
